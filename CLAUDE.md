# Geezword.com — Portfolio Hub

## Design

Design system: see ./DESIGN.md. Mode: **Hub**.

Read DESIGN.md before any UI code. Tokens are non-negotiable. Hub mode = editorial, gallery-like, parchment with indigo + saffron. Imagery dominant.

Hub-specific:
- This is the portfolio hub linking to Learn, Play, and Liturgy properties. Visually does not adopt any of their modes.
- Per-character color-music encoding (§3b) is NOT used in chrome — Hub uses the §3a foundational palette only.
- Imagery: real diaspora community, Ge'ez characters as visual objects, Ethiopian/Eritrean landscape sparingly. No stock photos, no kente, no Maasai imagery.
- Tone is editorial, not promotional.
